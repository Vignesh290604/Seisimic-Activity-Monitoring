# Seisimic-Activity-Monitoring
This project "IoT-Enabled pH & EC Monitoring for Intelligent Fertigation in Greenhouses," presents a state of the art, smart, automated nutrient-management system designed to maintain optimal soil and irrigation conditions for high-value crops. Manual nutrient regulation is often inefficient and prone to human error, leading to reduced yield and increased resource consumption. This system addresses these challenges by continuously measuring pH (acidity/alkalinity) and Electrical Conductivity (EC) (dissolved nutrient concentration) in the irrigation or hydroponic solution using dedicated sensors. The data is collected by an IoT Microcontroller (e.g., ESP32) and transmitted in real-time to a secure Cloud Dashboard for continuous monitoring and historical trend analysis accessible via web or mobile devices. The core innovation lies in the closed-loop automated control. When monitored pH or EC values drift outside the preset crop-specific optimal ranges, the microcontroller immediately triggers dosing pumps to inject corrective agents. This achieves precise fertigation without manual intervention. This design significantly reduces fertilizer waste, prevents nutrient imbalances, minimizes water consumption and through the cloud platform's data and potential predictive analytics, supports proactive maintenance and adaptive crop-stage adjustments. The implementation enables data-driven greenhouse management to improve crop yield, quality and overall agricultural sustainability. 
Farming is one of the tough and unpredictable work since its mostly depends upon climate and seasons. Traditional Farming consumes most of the time and effort. Farmers use different types of farming methods based on crop, season and fertility of the soil. Analysing all of this factors is a tough job, if there is any mistake the whole crop may get damage. Watering this Crop is also essential for time to time. Some of the crops may need more Water than other crops and some plants won’t even grown in a certain location. So, in this project focuses on developing a smart agricultural system that automatically monitors and controls nutrient levels required for plant growth. In greenhouse farming, maintaining the correct balance of nutrients, water, and environmental conditions is essential for achieving high crop productivity and quality. Traditional methods rely on manual monitoring, which can be time-consuming, inaccurate, and inefficient.
In this system, sensors such as pH sensor, Electrical Conductivity sensor, soil moisture sensor, temperature and humidity sensor, and LDR are used to continuously measure important environmental and nutrient parameters. These sensors are connected to an ESP32 microcontroller, which collects the sensor data and processes it using its built-in Analog-to-Digital Converter. The microcontroller compares the sensor readings with predefined threshold values suitable for plant growth.
The collected data is then transmitted through Wi-Fi to a cloud platform such as ThingSpeak, where it is stored and displayed in the form of graphs and real-time dashboards. This allows farmers or greenhouse operators to monitor the system remotely using a mobile phone or computer. When the measured pH or EC values move outside the desired range, the microcontroller automatically activates DC water pumps through relay modules to inject nutrient solution, fresh water, or corrective agents. This automatic adjustment helps maintain Proper nutrient balance in the irrigation system without manual intervention.
Overall, this IoT-based system enables real-time monitoring, automated nutrient control, efficient water and fertilizer usage, and improved crop productivity, making it a valuable solution for modern precision agriculture and sustainable greenhouse farming.
