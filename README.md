# Seisimic-Activity-Monitoring
The project titled "Seismic Activity Monitoring Through USGS API Integration in MATLAB" presents an automated system for retrieving, analyzing, and visualizing earthquake data using the United States Geological Survey (USGS) web services. The primary objective is to study the temporal and spatial patterns of seismic activity across different geologically active regions, including California, Japan, and Chile. Leveraging the USGS API, the system efficiently fetches earthquake data within specified time ranges and magnitude thresholds. In the event of connectivity or data access issues, synthetic data is generated to ensure uninterrupted analysis. The downloaded data undergoes statistical processing to compute critical seismic parameters such as the b-value and a-value based on the Gutenberg-Richter law. The analysis includes magnitude-frequency distributions, spatial b-value mapping through grid-based regional evaluations, and temporal b-value trends using sliding time windows. Results are visualized through MATLAB-generated plots and stored for further comparative studies across different regions. This integrated framework enhances understanding of seismic behavior and supports seismic hazard assessment by providing insights into earthquake recurrence patterns, regional stress conditions, and seismicity variations over time.
